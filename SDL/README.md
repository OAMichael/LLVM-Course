### Simple graphics sample

#### Preresiquites

You need [SDL](!https://github.com/libsdl-org/SDL). For linux you can install:

```console
sudo apt install libsdl2-dev
```

#### How to build and run

Being in folder SDL:
```console
mkdir build
cd build
cmake ..
make
```

Now run:
```console
./FreeStar.exe
```
